Answers to Qustions (10 March 2025):

When do you use a table?

The use of tables comes in various different reasons, however not all the time. If you are going to present the data with more than 1 columns and rows, then you should definitely consider using the table in HTML because it will help you organize the data efficiently.

When do you not use a table?

If you only need columns or rows ONLY, there are possible ways to achieve this efficiently and table aren't that useful for you to organize data effectively.

When do you need to do extra work for table accessibility?

You have to make sure that the table are accessible to all individuals. Making sure that they are labeled accurately will help you achieve the accessibility of the page and its table. You have to do extra work for table accessibility if there are critical data that should be labeled really well to avoid being sued.
