Answers to Questions:

1. Why is the viewport meta tag important?

The viewport meta tag in the HTML is found in the uppermost part of the HTML or inside the head tag. This means that every HTML format nowadays has this feature, this is very significant in the spacing behavior of the page. The viewport meta tag controls and manipulates the spacing varying in different devices such as desktop computers, laptops, smartphones and other devices that has access to view a certain website.

2. How can you adjust the CSS for printing?

The most efficient way to adjust the CSS for printing is first, create another CSS file, naming it as what you prefer. After creating the CSS File, link it in the html or inside the header alongside the CSS File link. After these procedures, you can now differentiate the two by adding a media=" " inserting one as screen and the other CSS File you just created as media="print". This will now be customizable in order for others to print your web efficiently while maintaining the aesthetic look of your webpage in the internet

3. When would you want to adjust the CSS for printing?

There are some visitors or viewers that sometimes wanted to print your webpagae. Some of them might want a clean ad readable look of your page to print, and some of them might want to become efficient to their ink usage in printing your page. These are the situations that you might need to consider adjusting your CSS File Print.
