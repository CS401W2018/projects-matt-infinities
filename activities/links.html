Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link is a link that will redirect an audience or visitor of a page into another page that has different domain, whereas the relative link is a link that will redirect a visitor of a page into a link that has a same domain name with the main page.

List the 3 rules with relative links.

  1. If it can be found on the same folder, use the format file.html
  2. If it is found in a child folder, use the format folder name/file.html
  3. If it is found in another folder, use the format ../projects/file.html

Do links always need to be inside a NAV tag?

No, it is not necessary to always put the links inside the <nav> tag. It can be applied in any part of the HTML other than <nave> tag
