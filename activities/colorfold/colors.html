Answer Questions:

List out the different ways to select an item.

There are 4 ways to select an item, Tag Name, Descendent Rule, Identification, and Class. All these 4 works in the HTML however, the most recommended ways to select is the Class selection method. It is highly flexible and customizable in using the CSS.

What is the difference between an ID and a Class?

The differences between ID and Class is the flexibility and functionality of the tag. The ID can be used by giving a certain tag an ID then editing its ID in the css, however the ID is limited, once you used it, it can't be used again. The class is flexible and functional because you can add a class name and multiple classes in a single tag.

What is the difference between a SPAN tag and a DIV tag?

DIV tag are commonly used in grouping each tags together, the SPAN tag on the other hand specifically built to customize a single tag only.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

A DIV tag differs from HEADER, MAIN, and FOOTER through its functionality. Whenever you use the HEADER, MAIN, and FOOTER, whatever you put inside these tags will affect your SEO score. The DIV on the other, is just a tag that will help you group the tags inside your HTML, it won't affect the SEO score.

What are the different ways to select a color?

There are various to select a color first, you can use the intensity of the color using its bits value. Every color has 1 byte equal to 8 bits which will give you 256 various intensities. You can use this method to choose a color. Second is the color code, you can use this method by entering # character before the color code so it will indentify that your going to use the color code. Third is the color wheel, whenever the box shows off in the HTML, click it and it ill redirect you to a color wheel where you can choose any color you prefer.
