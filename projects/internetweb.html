Internet

It is the interconnection of billions of devices that communicate with each other. This interconnection made our lives easier in sharing our information in a daily basis.

Web

Unlike to the Internet where it connects different devices, Web on the other hand connects documents, files, and pages. These are the works of an individual inside the computer and published it to the public for them to access it.

How are the Internet and Web different?

The difference of this two comes from their nature. The internet interconnects devices such as, computers, mobile phones, and other devices while the web interconnects someone's work in the internet such as pages, documents, and files.

ASCII and Unicode

ASCII is a set of computer codes and characters which mainly originated and develop by America. They always have an English based characters and number codes whereas the Unicode covers various types of characters and number codes depending on the language of origin.

What is UTF-8?

It is a standard on how characters are encoded into patterns and the computer will convert it into communication or information. It is widely used nowadays because it can encode both characters inside and outside the ASCII which is the Unicode.

URL

A URL or Uniform Resource Locator is an address to locate a certain webpage, file, image, video or document. It will grant you the access to a specific location.

Domain Name

Domain name is a readable name of a webpage. Computers that we have are communicating using IP addresses which aren't that understandable for humans, in order to understand such addresses, Domain name will convert those addresses into something readable.

HTTP

HTTP stands for Hypertext Transfer Procotol which an Internet protocol specific to the format of a message being interconnected to the client and the device.

HTTPS

HTTPS is not that far different from the HTTP, the only difference is that HTTPS communicates and formatted into a very secured way than HTTP alone.

IP Address

IP Address are codes of a particular address in the internet. This can be used to track a unique device or page to be able to access it. It can also serve as a location of a device or webpage.

DNS

It has a similarity to Domain Name. Only thing it differs is that DNS is a system. It has protocols and orders to follow.

W3C Recommendations

If the World Wide Web is another platform of the world, it should have its own governance. It is the work of W3C, it governs the World Wide Web so that it will remain operable, accessible, and user-friendly for everyone.

XML, XHTML, HTML, HTML5

XML is used to tranport and store data, XHTML is used to design and develop standards of a webpage, HTML is the tool to improve and develop the aesthetics of a wabpage, and lastly, the HTML 5 is the latest version of HTML nowadays, it grants you the latest and newest techniques in web development.
